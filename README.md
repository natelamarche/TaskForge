# TaskForge

TaskForge is a small command-line task manager built with **TypeScript** and **Node.js**.  
The goal of this project is not to build a complex app, but to learn the fundamentals of TypeScript and Node properly before moving on to Express, NestJS, and AWS.

## Why This Project Exists

This project is designed to teach the core skills needed for backend development:

- TypeScript types, interfaces, classes, and modules
- Node.js file system operations
- Async/await
- JSON data storage
- Error handling
- npm scripts
- Clean project structure
- Basic service-based architecture

The project intentionally starts without a web framework so the underlying Node.js and TypeScript concepts are easier to understand.

## Features

TaskForge supports basic task management from the command line:

- Add a task
- List all tasks
- Complete a task
- Delete a task
- Save tasks to a local JSON file
- Load tasks when the program starts

## Tech Stack

- TypeScript
- Node.js
- npm
- JSON file storage
